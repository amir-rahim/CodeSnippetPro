# CodeSnippetPro

CodeSnippetPro is a powerful tool designed to help developers manage and share their code snippets efficiently.

## Features

- **Snippet Management**: Organize your code snippets with tags and categories.
- **Search Functionality**: Quickly find the snippets you need with advanced search options.
- **Sharing**: Easily share your snippets with others.
- **Syntax Highlighting**: Supports syntax highlighting for multiple programming languages.

## Getting Started

To get started with the Blazor project, follow these steps:

1. Ensure you have the .NET SDK installed. You can download it from [here](https://dotnet.microsoft.com/download).
2. Clone the repository:
    ```bash
    git clone https://github.com/amir-rahim/CodeSnippetPro.git
    ```
3. Navigate to the project directory:
    ```bash
    cd CodeSnippetPro
    ```
4. Restore the dependencies:
    ```bash
    dotnet restore
    ```
5. Run the project:
    ```bash
    dotnet run
    ```

## Project Structure

The project structure is as follows:

- `Pages/`: Contains the Blazor components.
- `wwwroot/`: Contains static files such as CSS, JS, and images.
- `Shared/`: Contains shared components and layouts.
- `Data/`: Contains data models and services.
